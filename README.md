# WebGIS_simple_practices
Some after-class technical practices in the WebGIS class (GEOG5870M).

Unit1_FirstLeafletMap is a simple web page that displays five markers located at different buildings within the University of Leeds campus. A polygon is also included to outline an area of interest on the campus. Both the markers and the polygon are equipped with pop-up information windows.

Unit4_leedsBluePlaques is a more advanced web page that presents a map showing the geographic locations of blue plaques in Leeds. Each marker includes a pop-up window with information about the corresponding plaque. The page also provides search functionality, allowing users to filter results by time and name.

Unit6_HurricaneMap is a web map that visualizes hurricane-related tweets. It displays hotspots and clustering patterns of tweet locations. Users can click on individual markers to view detailed tweet content. Additionally, summary statistics are shown below the map, including the total number of tweets and the top five locations with the highest tweet density.
